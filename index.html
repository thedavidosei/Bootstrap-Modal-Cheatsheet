Components

Bootstrap modals should include the modal header, modal body (required for padding), and modal footer (optional). Always include in your modal headers dismiss actions whenever possible, or provide another explicit dismiss action.

Modal Example

To create a modal start by creating a div class="modal" tabindex="-1" role="dialog" .
Add add a div with the class div class="modal-dialog" role="document".
Add two more divs div class="modal-content" and div class="modal-header"
Now add your modal title. h5 class="modal-title">
Next add the code for the close-icon to dismiss the modal once the user is done button type="button" class="close" data-dismiss="modal" aria-label="Close" span aria-hidden="true" ×
Add a div for the modal's body and enclose the text with a p tag class="modal-body"
In the modal footer add the buttons necessary Click on the button to open the modal.
